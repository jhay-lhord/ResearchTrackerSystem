1. make sure you are inside in the directory ResearchTrackerSystem-v2, if not cd ResearchTrackerSystem-v2
2. then activate the virtual environment  by typing source venv/Scripts/activate on Linux or MacOS and for Windows use venv/bin/activate
3. after activating the virtual environment, install the required dependencies by running the command pip install -r requirements.txt
4. after that run the server  using python manage.py runserver.
